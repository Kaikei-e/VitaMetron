# 概日リズム（サーカディアンリズム）指標の導入と Circadian Health Score (CHS)

## ADR's STATUS

採用済み (2026-04-08)

## CONTEXT

VitaMetron は VRI (Vitality Recovery Index) で HRV・心拍・睡眠・呼吸数など 6 指標から回復度を定量化しているが、概日リズム（サーカディアンリズム）の健全性を直接評価する仕組みがなかった。

AHA 2025 Scientific Statement (Circulation) は概日リズムの乱れが肥満・2 型糖尿病・高血圧・心血管疾患のリスク増加と関連すること、不規則な睡眠タイミングが睡眠時間以上に強い予測因子となるケースを報告している。CosinorAge 研究 (npj Digital Medicine, 2024) は、ウェアラブル加速度計データからのコサイナー分析パラメータで生物学的年齢を推定できることを示した。

Fitbit から既に収集済みの HR Intraday（1 分解像度, 90 日保持）、Sleep Stages（30 秒解像度, 90 日保持）、Daily Summaries（無制限保持）で概日リズム分析が可能であり、追加の API 呼び出しは不要。

## DECISION MAKING

4 つの概日リズム指標群を既存データから計算し、5 次元の Circadian Health Score (CHS) として統合する。

**1. HR Cosinor Analysis (Cornelissen, 2014)**
24h HR データにコサイナーモデルを最小二乗法でフィットし、MESOR（リズム調整後平均 HR）、Amplitude（振幅）、Acrophase（ピーク時刻）を抽出。最小 720 サンプル（50%）必要。

**2. Non-Parametric Rest-Activity Rhythm (Van Someren et al., 1999)**
過去 7 日間の HR 時間帯パターンから IS（日間安定性, 0-1）、IV（日内断片化, 0-2）、RA（相対振幅, 0-1）、M10/L5 を算出。

**3. Sleep Timing Metrics (Roenneberg et al., 2004)**
睡眠中間点、中間点の円環標準偏差（ばらつき）、平日/休日差（Social Jetlag）を計算。

**4. Nocturnal HR Dip (Ohkubo et al., 2002)**
昼間（10-22 時）平均 HR と夜間睡眠中平均 HR の差。正常範囲 10-20%。

**CHS 複合スコア**: Rhythm Strength, Rhythm Stability, Rhythm Fragmentation, Sleep Regularity, Phase Alignment の 5 次元を VRI と同じ robust Z-score + tanh マッピングで 0-100 に統合。Phase Alignment は最適値 15% からの距離で両方向にペナルティ。Social Jetlag は表示のみ（Midpoint Variability と相関が高いため複合スコアには含めない）。

検討した代替案:
- Steps ベースのコサイナー → HR Intraday の方が rest-activity パターンの代理指標として優れるため不採用
- CHS を VRI に即時統合 → ベースライン安定に 14 日以上必要なため、まず独立スコアとして運用
- Social Jetlag を 6 番目の次元に追加 → Midpoint Variability と r > 0.8 で二重カウントになるため不採用

### PROS

- 既存データのみで概日リズムの健全性を定量評価（追加 API コール不要）
- 医学文献に基づく 5 次元の複合スコア
- VRI と同じスコアリング方式で一貫性がある
- 24h ポーラーチャートと睡眠中間点タイムラインで視覚的にパターンを把握可能

### CONS, TRADEOFF

- HR Intraday の 90 日保持制限により、長期トレンドは circadian_scores テーブルの集計値に依存
- CHS ベースライン安定に 14 日以上必要（cold start 期間中は confidence が低い）
- HR ベースの NPAR 指標は加速度計ベースのアクチグラフィーとは直接比較不可
- Social Jetlag は曜日ベースの自動判定であり、シフトワーカーには不正確
